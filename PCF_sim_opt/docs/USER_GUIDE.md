## 시스템 개요

### 무엇을 하는 시스템인가요?

배터리 제조 과정의 **탄소 배출량을 최소화**하는 최적의 자재 구성을 찾는 시스템입니다.

**주요 기능**:
- 양극재 원소별 최적화 (Ni, Co, Mn, Li)
- 음극재 Composition 최적화 (Natural vs Artificial Graphite)
- RE100 적용 최적화 (재생에너지 사용)
- 파레토 최적화 (탄소 vs 비용 트레이드오프)

### 시스템 구성

```
1. PCF 시뮬레이터 → 2. 최적화 엔진 → 3. 결과 분석
```

## 시작하기

### 1단계: 시스템 접속

1. 웹 브라우저를 열고 시스템 URL에 접속
2. 사이드바에서 **"PCF 시뮬레이터"** 선택

### 2단계: 데이터 준비

필요한 입력 데이터:
- **배터리 사양**: 용량, 에너지 밀도
- **자재 정보**: BOM (Bill of Materials)
- **배출계수**: 자재별 탄소 배출계수

## PCF 시뮬레이션

### PCF란?

**Product Carbon Footprint**: 제품(배터리)의 전체 생산 과정에서 발생하는 총 탄소 배출량

### 시뮬레이션 실행

1. **양극재 세부설정 탭**
   - 양극재 종류 선택 (NCA, NCM811, NCM622, LFP)
   - 각 양극재의 원소 구성 비율 확인

2. **시나리오 설정 탭**
   - 배터리 용량 입력 (예: 75 kWh)
   - 에너지 밀도 입력 (예: 250 Wh/kg)
   - 양극재 타입 선택

3. **시뮬레이션 실행**
   - "🚀 PCF 시뮬레이션 실행" 버튼 클릭
   - 결과 대기 (수초 소요)

### 시뮬레이션 결과 확인

**결과 탭**에서 다음 정보 확인:
- **총 탄소 배출량** (kgCO2eq)
- **자재별 배출량 기여도**
- **배터리 kWh당 배출량**

## 최적화 실행

### 최적화 흐름

```
데이터 로딩 → 제약조건 설정 → 최적화 실행 → 결과 확인
```

### 1단계: 데이터 로딩

**최적화** 페이지로 이동:

1. **데이터 로딩** 탭 선택
2. "📂 데이터 로드" 버튼 클릭
3. 로딩된 자재 목록 확인
   - ✅ 양극재 (Cathode Active Material)
   - ✅ 음극재 (Anode Active Material)
   - ✅ 전해액 (Electrolyte)
   - ✅ 분리막 (Separator)
   - 기타 자재

### 2단계: 제약조건 설정

**제약조건 설정** 탭에서 최적화 옵션 선택:

#### 비용 제약조건

**모드 선택**:
- **Zero-Premium Baseline**: 추가 비용 없이 최적화 (기본값)
- **Absolute Limit**: 비용 절대값 제한 (예: $5,000 이하)
- **Premium over Baseline**: 베이스라인 대비 추가 비용 제한 (예: +10%)

⚠️ **중요**: 파레토 최적화에서는 비용 제약이 적용되지 않습니다.

#### Feature Options (고급 기능)

1. **재활용 메탈 사용**
   - 양극재 원소에 재활용 메탈 적용
   - 비활성: 신재 메탈만 사용
   - 활성: 재활용 메탈 최적 비율 탐색

2. **저탄소 메탈 사용**
   - 저탄소 공정으로 생산된 메탈 사용
   - 비활성: 일반 메탈만 사용
   - 활성: 저탄소 메탈 최적 비율 탐색

3. **사이트 변경 허용**
   - 자재 공급지 변경 허용
   - 비활성: 고정 사이트
   - 활성: 최적 사이트 탐색

### 3단계: 최적화 실행

**최적화 실행** 탭에서:

#### 단일 목적 최적화

1. 목적 함수 선택: **"탄소 최소화"**
2. 솔버 선택: **GLPK** (기본, 빠름) 또는 **IPOPT** (고급)
3. "🚀 최적화 실행" 버튼 클릭

**소요 시간**: 수초 ~ 수십초

#### 파레토 최적화 (추천)

파레토 최적화는 **탄소 배출과 비용의 트레이드오프**를 탐색합니다.

**방법 선택**:

1. **Weighted Sum** (가중치 합)
   - 가중치를 변화시켜 프론티어 탐색
   - 빠르고 안정적
   - 포인트 수: 5~20개 (기본: 10개)

2. **Epsilon-Constraint**
   - 비용 상한을 변화시켜 탐색
   - 균일한 프론티어 생성
   - Epsilon 수: 5~20개 (기본: 10개)

3. **NSGA-II** (유전 알고리즘)
   - 다양한 해 탐색
   - 비선형 문제에 강함
   - 세대 수: 20~100 (기본: 50)
   - 인구 크기: 20~100 (기본: 50)

**실행**:
1. 방법 선택
2. 파라미터 조정 (선택)
3. "🎯 파레토 최적화 실행" 버튼 클릭
4. 진행 상황 확인 (진행률 표시)

**소요 시간**: 
- Weighted Sum / Epsilon-Constraint: 수십초 ~ 수분
- NSGA-II: 수분 ~ 수십분

## 결과 분석

### 결과 확인 탭

**최적화** 페이지의 **결과 확인** 탭에서 결과를 분석합니다.

### 파레토 프론티어 이해하기

**파레토 프론티어**는 탄소 배출과 비용 측면에서 동시에 개선할 수 없는 최적 해의 집합입니다.

```
비용 ($)
  ↑
  │     ● ← 높은 비용, 낮은 탄소
  │   ●
  │ ●
  │● ← 낮은 비용, 높은 탄소
  └────────────→ 탄소 배출 (kgCO2eq)
```

**선택 방법**:
- **탄소 중시**: 프론티어 왼쪽 포인트 선택
- **비용 중시**: 프론티어 오른쪽 포인트 선택
- **균형**: 프론티어 중간 포인트 선택

### 포인트 상세 정보

파레토 프론티어에서 포인트를 클릭하면 상세 정보 확인:

1. **요약 지표**
   - 총 탄소 배출량
   - 총 비용
   - 최적화 방법 (가중치/epsilon)

2. **양극재 최적화 결과**
   - 원소별 비율 (Ni, Co, Mn, Li)
   - 신재 / 재활용 / 저탄소 메탈 비율
   - 양극재 최적 배출계수

3. **음극재 최적화 결과** (Phase 3 추가)
   - Natural Graphite 비율
   - Artificial Graphite 비율
   - 음극재 최적 배출계수
   - 71.7% 감축 잠재력 확인

4. **자재별 상세 결과**
   - 각 자재의 최적 배출계수
   - RE100 적용 비율
   - 감축률 (Element-level + RE100)

### 시나리오 저장 및 비교

**시나리오로 저장**:
1. 원하는 포인트 선택
2. "💾 시나리오로 저장" 버튼 클릭
3. 시나리오 이름 입력

**시나리오 비교**:
1. **시나리오 비교** 탭으로 이동
2. 저장된 시나리오 선택 (2개 이상)
3. 비교 차트 확인
   - 탄소 배출량 비교
   - 비용 비교
   - 자재별 기여도 비교

## 최적화 결과 해석

### 양극재 원소별 최적화

**목표**: Ni, Co, Mn, Li 원소에 대해 재활용재/저탄소 메탈 비율 최대화

**결과 예시**:
```
Ni: 신재 20% / 재활용 50% / 저탄소 30%
Co: 신재 30% / 재활용 40% / 저탄소 30%
Mn: 신재 40% / 재활용 60% / 저탄소 0%
Li: 신재 50% / 재활용 50% / 저탄소 0%
```

**해석**:
- Ni는 재활용재 사용이 가장 효과적
- Co는 균형 있는 구성
- Mn, Li는 재활용재 위주

### 음극재 Composition 최적화

**목표**: Natural Graphite 비율 최대화 (배출계수가 71.7% 낮음)

**결과 예시**:
```
Natural Graphite: 85%
Artificial Graphite: 15%
```

**해석**:
- Natural Graphite 우선 사용으로 대폭 감축
- Artificial은 성능 요구사항에 따라 최소 사용

### RE100 적용

**Tier1** (최종 제품 제조):
- 양극재: CAM 제조
- 음극재: Anode 제조

**Tier2** (전구체 제조):
- 양극재: pCAM 제조
- 음극재: Graphite 전구체

**RE100 비율**:
- Tier1: 30% (예시)
- Tier2: 10% (예시)

**총 감축 효과** = (Element/Composition 감축) × (1 - RE100 감축)

## 실전 사용 예시

### 예시 1: 저탄소 배터리 설계

**목표**: 탄소 배출량 최소화

**단계**:
1. PCF 시뮬레이터에서 베이스라인 계산
2. 최적화 > 파레토 최적화 (Epsilon-Constraint)
3. 프론티어 왼쪽 포인트 선택 (최소 탄소)
4. 양극재/음극재 최적 구성 확인
5. 시나리오로 저장: "최소_탄소_설계"

**예상 결과**:
- 양극재: 재활용재/저탄소 메탈 최대 비율
- 음극재: Natural Graphite 100%
- RE100: Tier1 30%, Tier2 10%
- 총 감축: 60~80%

### 예시 2: 비용 제약 하 최적화

**목표**: $5,000 이하 비용으로 최대 감축

**단계**:
1. 제약조건 설정 > 비용 제약 > Absolute Limit: $5,000
2. 단일 목적 최적화: 탄소 최소화
3. 결과 확인

**예상 결과**:
- 비용 정확히 $5,000 (또는 그 이하)
- 해당 비용 내 최대 감축
- 일부 고가 옵션 제외 (저탄소 메탈 등)

### 예시 3: 트레이드오프 분석

**목표**: 다양한 탄소-비용 조합 탐색

**단계**:
1. 파레토 최적화 (Weighted Sum, 15개 포인트)
2. 프론티어 전체 확인
3. 관심 구간 (예: 중간 비용대) 포인트 3개 선택
4. 시나리오로 저장: "옵션_A", "옵션_B", "옵션_C"
5. 시나리오 비교 탭에서 상세 비교

**의사결정**:
- 옵션_A: 높은 감축, 높은 비용
- 옵션_B: 중간 감축, 중간 비용 (균형)
- 옵션_C: 낮은 감축, 낮은 비용

## 자주 묻는 질문

### Q1. 최적화가 실패했습니다. 왜 그런가요?

**원인**:
1. 제약조건이 너무 엄격함 (실현 불가능)
2. 솔버 오류 (GLPK/IPOPT)
3. 데이터 오류

**해결 방법**:
1. 제약조건 완화 (비용 상한 증가 등)
2. 다른 솔버 시도
3. 데이터 로딩 재시도

### Q2. 파레토 포인트가 적게 나옵니다.

**원인**:
- Epsilon-Constraint에서 일부 epsilon 값이 실현 불가능

**해결 방법**:
- Epsilon 범위 조정 (더 넓게 설정)
- Weighted Sum 방법 사용
- NSGA-II 사용 (더 다양한 해 탐색)

### Q3. 음극재 최적화 결과가 안 보입니다.

**확인 사항**:
1. 데이터에 음극재(Anode) 자재가 포함되어 있는가?
2. Natural Graphite와 Artificial Graphite가 모두 있는가?
3. 최적화가 정상 완료되었는가?

**비고**: Phase 3 이후 버전에서만 지원됩니다.

### Q4. RE100 적용 비율은 어떻게 결정되나요?

RE100 적용 비율은 **데이터로부터 자동 계산**됩니다:
- Tier1: Formula 자재의 에너지 비율 × RE100 계수
- Tier2: Formula 자재의 에너지 비율 × RE100 계수

현재는 고정값이지만, 향후 최적화 변수로 확장 가능합니다.

### Q5. 시나리오를 저장했는데 어디서 확인하나요?

**시나리오 비교** 탭에서 저장된 모든 시나리오를 확인할 수 있습니다.

### Q6. 결과를 엑셀로 내보낼 수 있나요?

현재 직접 엑셀 내보내기는 지원하지 않습니다. 
웹 페이지의 표를 복사하여 엑셀에 붙여넣을 수 있습니다.

향후 CSV/Excel 내보내기 기능 추가 예정입니다.
